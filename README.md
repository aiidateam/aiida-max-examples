AiiDA MaX Examples
-------------------

Examples of how to run [MaX flagship codes](http://www.max-centre.eu/flagship-codes/)
through the [AiiDA python framework](http://www.aiida.net/).

Usage
-----

```
workon aiida
verdi daemon start
./run.py -c siesta-4.0.1@localhost -s TiO2
```
